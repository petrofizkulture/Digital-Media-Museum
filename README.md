# Universal ID Numbering System (UIDNS)
UIDNS is a set of a customizable 5-digit ID of any variety
We used our own exclusive for this application, let me show you what are they!
## 0x - Full Versions
## 1x - Lite, Non-Full Release Versions
## 2x - Sequels
## 3x - Apple Arcade / Apple Device Exclusive
## 4x - CHN Region Applications or CHN Variants
## 5x - Android Market / Google Play Exclusives
## 6x - App extensions / Different Hardware Only
To avoid confusion between different apps, you must select a category from the following:
O7 - Outfit7
Wii - Nintendo Wii
WW - WiiWare
DS - Nintendo DS
DW - DSiWare
3DS - Nintendo 3DS
CTRSA - 3DS System Applications
AA - Apple Arcade/Apps
AM - Android Apps (using AM for Android Market which is now Google Play Store)
HW - Different Hardware Exclusive Apps
